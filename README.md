## Problem Statement

Payslip fraud is a growing problem. People edit PDF payslips 
and submit fake salary proofs to:

- **Banks** — for home loans and personal loans
- **Landlords** — for house rental agreements  
- **Visa Offices** — for foreign travel applications
- **Employers** — for inflated salary negotiations
- **Colleges** — for scholarships and fee concessions
- **Courts** — to hide real income in legal cases

There is currently **no way to verify** a payslip without 
calling the company directly. LedgerShield solves this.

---

## Solution

Paychain generates a **SHA-256 cryptographic hash** of 
every payslip and locks it permanently on the **Ganache 
blockchain** using a **Solidity smart contract**. 

Anyone — bank officer, landlord, visa officer — can paste 
the hash on our public verifier page and instantly know if 
the payslip is **GENUINE or FAKE** in under 1 second.

## How MetaMask is Used

MetaMask is a **crypto wallet browser extension** that acts 
as the HR's digital identity on the blockchain.


HR opens PayChain website
         ↓
Clicks "Connect Wallet"
         ↓
MetaMask popup opens — HR approves
         ↓
HR fills payslip form and clicks Generate
         ↓
MetaMask popup shows transaction details
HR clicks "Confirm"
         ↓
MetaMask signs the transaction with
HR's private key — proving it was
genuinely authorized by HR
         ↓
Signed transaction sent to Ganache blockchain
         ↓
Smart contract stores hash permanently


Without MetaMask, anyone could generate fake payslips. 
MetaMask ensures **only authorized HR** can generate payslips.

## How It Works

### Generating a Payslip (HR Side)
1. HR connects MetaMask wallet to PayChain
2. HR fills employee salary details on HR Dashboard
3. System calculates SHA-256 hash of all payslip data
4. MetaMask signs the transaction — HR confirms in popup
5. Hash stored permanently on Ganache blockchain
6. Full payslip details saved in MongoDB database
7. PDF generated with blockchain hash printed at bottom
8. Employee receives payslip with unique verification hash

### Verifying a Payslip (Anyone)
1. Open Public Verifier page — no login needed
2. Enter Employee ID and paste hash from payslip PDF
3. System checks blockchain record in real time
4. Result appears in under 1 second:
   - **GENUINE** — hash matches blockchain record
   - **FAKE** — hash does not match — document tampered

### How Fake Detection Works

Original payslip hash stored on blockchain:
abc123xyz...

Someone edits PDF — changes salary from
Rs.25,000 to Rs.75,000

System recalculates hash from edited data:
zyx987abc... ← completely different!

Hash does not match blockchain record
System shows: FAKE ❌
